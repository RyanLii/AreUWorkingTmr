# Contributing

Thanks for your interest in contributing.

## Development setup

1. Install Xcode (latest stable) and Swift 6 toolchain.
2. Generate the project:

```bash
xcodegen generate
```

3. Run core tests:

```bash
swift test
```

## Pull request checklist

- [ ] Keep changes scoped and clearly described.
- [ ] Add/update tests for core logic changes.
- [ ] Ensure `swift test` passes.
- [ ] Update README/docs if behavior changes.

## Code style

- Follow existing Swift naming/style conventions.
- Prefer small, focused commits.
- Keep user safety copy clear and non-judgmental.
